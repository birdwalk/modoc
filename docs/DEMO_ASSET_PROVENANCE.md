# Demo Asset Provenance

All included demo models are programmatically generated by `apps/api/modeldoctor_api/demo_models.py` for MODOC.

Generated assets:

- Clean cube.
- Cube with missing face.
- Mesh with flipped normal.
- Mesh with non-manifold edge.
- Duplicate-vertex mesh.
- Multiple disconnected shells.
- Tiny floating component.
- Self-intersection candidate.
- High-poly decimation candidate.
- Thin-wall candidate.
- Multi-defect demonstration model.

No third-party model assets are bundled.
