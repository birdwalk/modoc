# Hackathon Demo

Three-minute flow:

1. Open MODOC.
2. Click **Load Demo Model**.
3. Show selected-use health score.
4. Rotate the model.
5. Select a boundary or non-manifold issue.
6. Read the deterministic explanation.
7. Click **Apply Safe Fixes**.
8. Compare before and after metrics.
9. Ask Copilot whether it is ready for 3D printing.
10. Download repaired OBJ.
11. Prepare and download PDF, JSON and CSV reports.

Talking point: MODOC behaves like a geometry health checker for 3D files, but defects come from deterministic code.
